# 🚀 Automated Website Testing Tool

## 📌 Overview

This project is designed to **automatically scan websites and identify common issues**, including:

* UI inconsistencies
* Broken links
* Missing or broken images
* Grammar and spelling issues
* Functional failures (buttons, forms, navigation)

It helps testers and developers quickly identify problems and generate a **clear bug report with screenshots**.

---

## 🎯 Features

* 🔍 Website scanning automation
* 📸 Screenshot-based bug reporting
* ❌ Broken link detection
* 🖼️ Image loading validation
* 📝 Grammar and spelling checks
* ⚙️ Functional testing (buttons, forms, navigation)
* 📊 HTML report generation with highlighted issues

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/rohanshukla1/For-Automate-Test-of-Website.git
```

Go to project folder:

```bash
cd For-Automate-Test-of-Website
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Usage

Run the scanner:

```bash
npm start
```

OR (based on your script):

```bash
node index.js
```
